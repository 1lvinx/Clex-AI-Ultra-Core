# Project Overview / 项目概览

## What is Clex-AI-Ultra Core?

Clex-AI-Ultra Core 是一个**多语言工程化核心版本 (Polyglot Core Edition)**，为 OpenClaw 提供 AI-assisted development 能力增强。

## Core Capabilities / 核心能力

| 能力 | 描述 |
|------|------|
| **Polyglot Engine** | Python/Node.js/Rust/Go 四语言协同 |
| **Workflow Orchestration** | 高阶编排与任务调度 |
| **Permission Control** | 细粒度安全控制 |
| **Agent Coordination** | 多 Agent 协作 |
| **MCP Integration** | Model Context Protocol 集成 |

## Why Polyglot? / 为什么多语言？

| 语言 | 职责 | 优势 |
|------|------|------|
| Python | Workflow orchestration | 简单语法、丰富生态 |
| Node.js | CLI & SDK | 最佳 CLI 框架 |
| Rust | Validation | 内存安全、编译时检查 |
| Go | Worker & parallelism | Goroutines、生产就绪 |

## Project Goals / 项目目标

1. **公开安全** - 无泄露，无凭证，无专有代码
2. **可扩展架构** - 模块化设计，易于扩展
3. **跨语言协同** - Python/Node.js/Rust/Go 协同工作
4. **工程化文档** - 完整的架构与使用文档

## Roadmap / 发展路线

- **v0.1.0-alpha** - Core 基础架构
- **v1.0.0-core** - 完整 Core Edition
- **v1.1.0-extended** - Extended Edition
- **v2.0.0-local** - Local/Private Edition
