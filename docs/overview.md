# Project Overview / 项目概览

## What is Clex-AI-Ultra Core?

Clex-AI-Ultra Core is an **Open Source Polyglot Core Edition** for AI-assisted development, providing enhanced capabilities to OpenClaw through workflow orchestration, permission control, and agent coordination.

## Core Capabilities

| Capability | Description |
|------------|-------------|
| **Polyglot Engine** | Python/Node.js/Rust/Go协同 |
| **Workflow Orchestration** | 高阶编排与任务调度 |
| **Permission Control** | 细粒度安全控制 |
| **Agent Coordination** | 多 Agent 协作 |
| **MCP Integration** | Model Context Protocol 集成 |

## Why Polyglot?

| Language | Responsibility | Status |
|----------|----------------|--------|
| Python | Workflow orchestration | MVP |
| Node.js | CLI & SDK | Skeleton |
| Rust | Validation | Skeleton |
| Go | Worker & parallelism | Skeleton |

## Project Goals

1. **Public-safe** - 无敏感信息，无凭证，无专有代码
2. **Extensible Architecture** - 模块化设计，易于扩展
3. **Cross-language Collaboration** - Python/Node.js/Rust/Go 协同工作
4. **Engineering Documentation** - 完整的架构与使用文档

## Roadmap

- **v0.1.0-alpha** - Core 基础架构 (Public Alpha)
- **v1.0.0-core** - 完整 Core Edition
- **v1.1.0-extended** - Extended Edition
- **v2.0.0-local** - Local/Private Edition
