# Project Overview / 项目概览

Clex-AI-Ultra 是一个为 OpenClaw 平台设计的 AI 工程增强技能集合。

## What is OpenClaw?

OpenClaw 是一个模块化 AI 开发框架，支持多种 AI 模型、工作流编排和技能扩展。

## What Are Skills?

Skills 是 OpenClaw 的核心扩展机制，每个 skill 是一个独立的模块，提供特定功能：

- 代码分析与生成
- 任务调度与分发
- CLI 命令封装
- MCP 协议集成
- Agent 团队协作

## Project Goals / 项目目标

1. 提供高质量的官方技能集合
2. 输出完整的工程化文档
3. 支持快速安装和零依赖使用
4. 保持安全合规的开源代码
