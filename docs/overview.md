# Project Overview

## What is Clex-AI-Ultra Core?

Clex-AI-Ultra Core is an **Open Source Polyglot Core Edition** for AI-assisted development, providing enhanced capabilities to OpenClaw through workflow orchestration, permission control, and agent coordination.

## Core Capabilities

| Capability | Description |
|------------|-------------|
| **Polyglot Engine** | Python / Node.js / Rust / Go combined |
| **Workflow Orchestration** | High-level orchestration & task scheduling |
| **Permission Control** | Fine-grained access control |
| **Agent Coordination** | Multi-agent collaboration & task dispatch |
| **MCP Integration** | Model Context Protocol integration |

## Why Polyglot?

| Language | Responsibility | Status |
|----------|----------------|--------|
| Python | Workflow orchestration | MVP |
| Node.js | CLI & SDK | Skeleton |
| Rust | Validation | Skeleton |
| Go | Worker & parallelism | Skeleton |

## Project Goals

1. **Public-safe** - No credentials, no private infrastructure, no unauthorized source code
2. **Extensible Architecture** - Modular design, easy to extend
3. **Cross-language Collaboration** - Python / Node.js / Rust / Go combined
4. **Engineering Documentation** - Complete architecture and usage documentation

## Roadmap

- **v0.1.0-alpha** - Core foundation (Public Alpha)
- **v1.0.0-core** - Complete Core Edition
- **v1.1.0-extended** - Extended Edition
- **v2.0.0-local** - Local/Private Edition
