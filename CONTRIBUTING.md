# Contributing / 贡献指南

Thank you for your interest in contributing to Clex-AI-Ultra!

## How to Contribute / 如何贡献

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**
4. **Run the validation script**
   ```bash
   python scripts/validate-skills.py
   ```
5. **Submit a Pull Request**

## Skill Contribution Guidelines / 技能贡献指南

All skills should:

- Have a clear purpose
- Include comprehensive `SKILL.md` documentation
- Follow the naming convention: `skill-name`
- Avoid hardcoded paths or credentials
- Be tested before submission

## Code Style / 代码风格

- Use consistent indentation
- Add comments for complex logic
- Follow existing patterns in other skills

## Questions? / 有问题？

Open an Issue or reach out to the maintainers.
